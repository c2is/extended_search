Features:
---------
The Extended Search module extends search module to make search on "regexp" condition in place of "like".


Installation:
------------
1. Download and unpack the Extended Search in "sites/all/modules".
2. Go to "Administer" -> "Modules" and enable the module.
2. Go to "Administer" -> "Configurations" -> "Search settings", enable the module and check the radio Extended Search in the frame "Default search module".
